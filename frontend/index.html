为什么现在可能还不能跑？

这个前端只是工作台。真正抓行情、算指标、调用 GPT 的地方在后端 API。你需要先部署或启动后端。

GPT API 在哪里设置

不是在这个网页里填，是在后端环境变量里设置。

1. 必填 Key

OPENAI_API_KEY=你的 key

2. 模型名称

OPENAI_MODEL=gpt-5.5

3. 兼容网关可选

OPENAI_BASE_URL=https://your-gateway/v1

4. 本地启动示例

cd backend && uvicorn main:app --reload --port 8000
